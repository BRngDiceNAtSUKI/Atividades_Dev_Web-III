## Descrição do Projeto

Este projeto é um **sistema web institucional** desenvolvido com **Node.js puro**, sem frameworks, demonstrando domínio de rotas, organização de arquivos e hierarquia de diretórios.  
O tema é inspirado em **Stalker Shadow of Chernobyl**, com produtos representando os **Produtos** da série e layout moderno em cards.

O sistema inclui:

- Página inicial (Home) estilizada com botões/cards de navegação.  
- Página institucional "Quem Somos".  
- Página de produtos com cards de réplicas, estatuas e Filme (Stalker Estátua, Detector de anomalias e Filme Stalekr).  
- Páginas individuais de cada produto, cada uma com card detalhado.  
- Página de erro 404 personalizada que retorna para a Home.  
- CSS unificado e moderno, com efeitos de hover e layout responsivo.
